# 🌿 Health Pilot

Health Pilot は、Apple Health・SOXAI・AI を活用して、
毎日の健康を「今日やるべき3つ」に変換する
AIパーソナルヘルスアシスタントです。

---

## Vision

健康データを見るアプリではなく、

健康になるために行動できるアプリ。

---

## MVP

- 健康スコア（100点）
- 今日のMission 3つ
- AIコメント
- 睡眠改善サポート
- Apple Health連携
- SOXAI連携
